Arrays is an ordered collections of values

e.g Array means "reeks". Examples below of where arrays are being used

  • list of comments on IG post
  • Collections of levels in a game
  • Songs in a playlist

      • Creating arrays

    • We use square brackets [] to denote a array
    • Don't use typeof to check if something is an array because it will say "object"
    • You can put anything in an array, numbers, strings and etc all together. You can even put an array within an array

Arrays are indexed

  • Each array has an index, just like that every character in a string is also indexed so each element has a corresponding index. starts at 0
  • Unlike a string you can change a character or/and element in an array by using the index let colors = ['rad', 'blue', 'yellow']
    colors[0] = 'red'
    so this means that the rad has changed into 'red' in the array. we can't do something like this with just a string value of a variable
