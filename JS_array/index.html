Arrays are indexed

  • Each array has an index, just like that every character in a string is also indexed so each element has a corresponding index. starts at 0
  • Unlike a string you can change a character or/and element in an array by using the index let colors = ['rad', 'blue', 'yellow']
    colors[0] = 'red'
    so this means that the rad has changed into 'red' in the array. we can't do something like this with just a string value of a variable
We can add and remove from the end of the array or we can add and remove from the beginning.

PUSH AND POP - END

push - add to end

pop - remove from end

SHIFT AND UNSHIFT - START

shift - remove from start

unshift - add to start

MORE METHODS

concat() - combining array method is used to merge to or more arrays. it doesn't change the exisiting arrays but instead returns a new array

prototype.concat(value0, value1, value2....)

includes() - it's a boolean method to see if a element is included in the array of not -- simple yes or no

indexOf() -- to see if an element is in an array or not and what the index it is. so it's more detailed than includeds(). -1 means its not present

reverse() -- you don't have to put something in the parentheses it just give you a reversed copy. it reverses the original array

slice() -- getting a slice of an array.

splice() -- this method doesn't give you a copy it will change the original array

sort() -- it is sorted in place and no copy is created, so the array is altered with the sorted values. by default it will be sort in ascending order BUT if there are any undefined valued they will automatically sorted to the back of the array. Use the compareFunction to sort them into proper numerical orderl. by using only sort() the numbers will be converted into strings and it will therefore only look at the first number and put them in ascending order.

OBJECT LITERALS

Objects are a collections of properties. They are a key-value pair. An object literal is a comma seperated lits of key-value pairs inside of curly braces. These values can be properties or functions. property = key + value {key:value} objects are unlike arrays not ordered!

How to acces an object literal

every key that we make in objects is converted to a string

if you want to acces an object literal you need to use square brackets and quotation marks let person = {
firstName: "Mick",
lastName: "Jagger" }
person["firstName]

you can also use the DOT syntax. eg person.firstName or person.lastName

firstname key is actually a string
